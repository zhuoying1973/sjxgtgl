# Archviz Biz Manager 项目进度固化手册 (2026-03-19)

## 1. 当前进度总结

### 已完成的核心重构 (极简垂直版)
- **数据库升级 (V2)**: 
    - 为 `projects` 表新增了 `tax_rate` (税率)、`ai_factor` (AI折收因子) 和 `ratio_scheme` (工序提成比例方案)。
    - 完成了历史项目的比例方案自动回填，确保数据一致性。
- **双向控费提成算法**:
    - **个人计件逻辑**: `提成 = (工作量 * 动态单价) * (AI因子 / 100)`。
    - **毛利保护逻辑**: 实现了根据项目经理设置的“工序比例方案”自动计算项目总提成上限。
- **极简调控台 (全局设置)**:
    - 实现了 `SystemSetting` 动态读取方法。
    - 在“财务设置”页面增加了全局单价调节（建模、渲染、后期）和默认比例方案 JSON 配置。
- **项目管理优化**:
    - 编辑页面增加了“税率”、“AI折收因子”以及“工序比例配置”的实时调控面板。
    - 后端路由 `project_update` 已全面兼容这些动态参数的保存。

### 界面组件
- **项目编辑页 (`project_edit.html`)**: 已集成 AI 调控因子。
- **财务设置页 (`settings_finance.html`)**: 已进化为“参数调控中心”。
- **提成计算底层**: 已打通“全局默认 -> 项目个性化”的优先级链路。
